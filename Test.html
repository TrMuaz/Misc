Sample Lesson Page with Canva Slides

Lesson: Introduction Slides

This page shows how a Canva presentation can be embedded inside a normal HTML lesson page hosted on GitHub Pages. Students can view and navigate the slides directly in the browser.

Slides (Canva Embed)

If the slides do not load, please check that the Canva design is set to be shared publicly or use the official embed code from Canva’s Share → Embed option.

Teacher Notes

You can add instructions for students here, such as which slides to review, reflection questions, or links to quizzes and worksheets.
